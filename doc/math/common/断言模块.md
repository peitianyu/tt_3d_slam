## 断言

```c++
// 参考c++断言模块实现, 使用#define实现, 这里使用LOG_ERROR打印
#define tt_assert(expr)                                                                         \
    if (!(expr)) {                                                                              \
        LOG_ERROR("assert failed: ", #expr, __FILE__, __FUNCTION__, __LINE__) << std::endl;     \
        exit(1);                                                                                \
    }
```
